# Hammurabi

> AI を活用したアプリケーション開発のためのロードマップシステム

---

## 概要

**Hammurabi** は、AI との対話を通じてアプリケーションの設計・実装・管理を効率的に進めるための「開発ロードマップ」システムです。

プロジェクト名は古代バビロニアの「ハンムラビ法典」に由来しており、明確なルールとガイドラインに基づいた開発プロセスを支援することを目指しています。

---

## 特徴（予定）

- **AI 駆動の開発支援**: AI との対話によるアーキテクチャ設計・コード生成をサポート
- **ロードマップ管理**: 開発ステップの可視化と進捗管理
- **ルールベース設計**: 一貫性のある開発ガイドラインの定義・適用

---
